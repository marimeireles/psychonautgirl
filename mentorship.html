mentorship



Hello! Are you interested in being a mentor for Women of Color Code, ReDI Berlin, Open Life Science or PyLadies Berlin community and have questions about it? Check this link and feel free to contact me.

Are you a member of one of these communities and would like to schedule a slot with me? Please try to use this link, if none of these time work for you, feel free to shoot me a message and we can figure something out!

I thought it'd be useful to come up with a list of things I can do for people. The mentoring is not restricted to these things, feel free to come up with your own needs and ideas, this is just to help you to think on things where I could be useful to you:

  • help you to write a cover letter for a job
  • build a linkedin profile
  • build a resume
  • build a personal website
  • help you to prepare for software engineer related job interview
    • conduct mock interviews
      • I can pretend to interview you! I'll use the STAR method to ask general questions. I can also prepare a code challange one if you wish (but these are really stressfull and annoying, so no stress if you don't want to do them)
  • can help you with a specific software topic on a session, check my main website and github if you're in doubt on what kinds of things I can mentor you on
  • help you to set goals and keep you on track to achieve your goals
    • as in have regular check-ins to see how you're doing and what do you need to keep going
    • or just change to a new goal because your current goal doesn't ressonate with you anymore
  • coworking together
    • we can meetup in berlin and cowork together. The idea for this is not that i'm following you in every step, but that we're coworking side by side and I can help you any time you're stuck
    • I work with open source and if you're interested I can show you how to contribute to open source to my project, check pyscript.net for the general idea and for the code github.com/pyscript/pyscript
    • if you're not in Berlin we can try something online, just book a session!
